Detailed Description

VoycePage is a simple updated version of the Felgo's Page component. It is used as a reusable page for multiple specific instances across the app.

Property Documentation

searchBarEnabled : bool

If set, search bar will be displayed for targetted AppListView in the current page


showAcitivtyIndicator : bool

If set, spinning activity indicator will be visible on top navigation bar
