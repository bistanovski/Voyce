Detailed Description

VoyceListView is a simple updated version of the Felgo's AppListView component.

Property Documentation

loadingText : string

If set, while data is loaded, this text will be shown as main content


refreshFuncHandler : var

Callback function handler, which is used as target to invoke on PullToRefreshHandler::refresh() signal
