Import Statement: import

Properties

Detailed Description

VoyceRightBarItem is a simple updated version of the Felgo's NavigationBarRow component. It sets ActivityIndicatorBarItem and IconButtonBarItem elements.

Property Documentation

activityIndicatorVisible : bool

If set, activity indicator in the navigation bar will be visible.
